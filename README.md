# mean-app

<p>Type</p>
<pre>npm install</pre>
<p>in server folder and in client/myApp folder</p>
<p>Then run</p>
<pre>node app</pre>
<p> in server folder and </p>
<pre>ng serve</pre>
<p>in client/myApp folder</p>
<p>(<em>I hope you've installed <a href="https://nodejs.org/uk/download/">Node.js and npm</a></em>)</p>
